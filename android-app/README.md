# DriveeUP Android (Kotlin + Jetpack Compose + MVVM)

Стартовый каркас мобильного приложения.

## Запуск
1. Открыть папку `android-app` в Android Studio.
2. Создать новый проект `Empty Compose Activity` с package `ru.driveeup.mobile`.
3. Заменить созданные файлы на файлы из `app/src/main/java/ru/driveeup/mobile`.
4. Добавить базовый URL API (`https://driveeup.ru/api`).

## Что уже заложено
- MVVM структура (`ui`, `domain`, `data`)
- Экран авторизации
- Разделение ролей DRIVER/PASSENGER
- Заготовка под Battle Pass и игры
